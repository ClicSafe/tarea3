# Ubicación de imágenes en Spring Boot

Para incluir imágenes en tu aplicación Spring Boot, deberías colocarlas en la carpeta de recursos estáticos. Spring Boot tiene una convención para organizar archivos estáticos como imágenes, CSS y JavaScript.

## Paso a paso:

1. Crear la estructura de carpetas para recursos estáticos
2. Colocar las imágenes en la ubicación correcta
3. Hacer referencia a las imágenes en tus plantillas HTML

### Estructura de carpetas recomendada

Esta carpeta es para almacenar las imágenes de la aplicación.
Puedes organizar las imágenes en subcarpetas si es necesario (por ejemplo: logos, backgrounds, icons, etc.)
